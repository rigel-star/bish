# bish - bytecode interpreter in Rust

## Quick Start
```console
$ cargo run
```

## Demo Program
```console
rakha x ma 5;
rakha y ma 5;
yadi chhaina (x barabar y) {
    dekhau "x barabar y chha";
    dekhau "\nx ko value:";
    dekhau x;
    dekhau "\ny ko value:";
    dekhau y;
}

dekhau x + y;
```
